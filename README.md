# iotled

 Example IOT: Spring Boot + Pi4j + MQTT Eclipse Paho + Raspberry Pi 3 + LED
 
 Needed software:
    - Spring Boot + Angularjs 4
    - Pi4j
    - Java 8
    - Maven
    - MQTT Broker 

mvn clean package and run the .jar on your Raspberrypi
